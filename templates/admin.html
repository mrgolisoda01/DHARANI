Create a new assessment

Type questions in Excel, save as CSV, and paste them below. Each learner gets a random shuffled subset.

CSV format (first row = header):
question,option_a,option_b,option_c,option_d,correct,category
option_c, option_d, category are optional. "correct" must be A, B, C, or D.

Example:
Glass bottle cost with tax?,₹4.54,₹12.91,₹15,₹30,A,P&L
